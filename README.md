# flask-app-ecs
Simple flask app to be run
1.  git clone https://github.com/LondheShubham153/flask-app-ecs.git
2.  ls -ll
3.  cd flask-app-ecs/
4.  vim Dockerfile
5.  ls -ll
6.  cat app.py
7.  vim app.py
8.  cat run.py
9.  docker build -t flask-app .
10. docker images
11. docker run -d -p 80:80 flask-app
